# Portfolio UI Kit

Recreation of the interactive hub of Mahesh Inder's Multiverse portfolio — focused on components, not the full 3D scene.

Covers:
- Magazine masthead (MAHESH / INDER) with reveal
- Compact logo + Paperwork/Live buttons + glass dropdown menu
- macOS-style Planet Dock
- Orbital nav (2D CSS version)
- Persona section (with bleed orb, project card, quote, skill tags)
- Starfield background layer

Open `index.html` for a click-thru demo. Hit the masthead letters once to reveal the planets, then click a dock item to navigate between personas.
